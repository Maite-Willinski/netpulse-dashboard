<h1 align="center">Monitorización, Orquestación y Resiliencia 👋 ✨</h1>

<p align="center">
  <strong>⚠️ Nota de Proyecto:</strong> NetPulse es un <strong>Dashboard de simulación técnica (Frontend Mock)</strong>. Todos los datos, métricas y estados mostrados son ficticios y han sido diseñados con fines educativos para demostrar habilidades en el desarrollo de interfaces de usuario para entornos SysOps y SecOps.
</p>

<p align="center">
  El proyecto fusiona una interfaz moderna "Cozy-Tech" con una lógica interactiva compleja, recreando de forma autónoma el comportamiento de un entorno de red real para validar la implementación de buenas prácticas en la maquetación y el control de estado.
</p>

## 🚀 Módulos Funcionales

- 🖥️ **SysOps Control:** Monitorización interactiva con datos simulados.
- 🐳 **Docker Engine Simulator:** Panel de control de estados para microservicios ficticios.
- 🔌 **LAN & ARP Mapper:** Generador de tablas de red con búsqueda predictiva.
- 🚨 **SecOps & Resilience:** Gestión de firewall y simulación de incidentes críticos.
